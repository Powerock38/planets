# Planets
NodeJS + HTML multiplayer game featuring randomly generated planets and spaceships

## How to install & run
1. `git clone https://github.com/Powerock38/planets`
1. `cd planets`
1. `npm install`
1. `node app`
1. Go to http://localhost:2000/
1. Profit !
